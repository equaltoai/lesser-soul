# Archived Untracked Content

This folder captures files that were untracked in `lesser-soul` on 2026-03-18 and were moved out of the active
working tree to keep the repo easier to review.

It also contains snapshots of the 2026-03-18 working-tree versions of tracked top-level files that were intentionally
archived before restoring the active copies from `HEAD`.

Layout:

- `root/` contains files that originally lived at the repository root.
- `docs/` contains files that originally lived under `docs/`.
- `roadmaps/` contains files that originally lived under `roadmaps/`.
- `tracked-working-tree/` contains archived snapshots of tracked files that had local edits.

The active top-level `ROADMAP.md` and `SPEC.md` files were restored from `HEAD` after their working-tree snapshots were
copied into `tracked-working-tree/`.
